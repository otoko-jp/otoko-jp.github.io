menuとは

フードデリバリーサービス。スマートフォンアプリから近隣のレストランや飲食店に注文すると、最短30分程度で届く。Uber Eatsと並ぶ国内大手のひとつで、対応エリアは東京・大阪・名古屋をはじめとする主要都市に広がっている。

深夜帯でも対応している店舗が多く、コンビニ系や弁当チェーンも充実している。Uber Eatsと比較して送料が低めに設定されているケースも多く、両方を使い分けているユーザーも多い。アカウント登録はメールアドレスのみで完結し、本人確認は不要だ。

特典の詳細

プロモーションコードを使うと、新規登録後の最初の8回の注文に割引が適用される。初回と2回目以降で割引額が異なる段階制になっている。

回数割引額最低注文金額
1回目1,200円OFF1,500円以上
2〜8回目700円OFF（各回）1,500円以上
合計最大6,800円OFF—

コードの有効期間は登録後31日以内に使用開始することが条件。31日を過ぎると未使用の割引は消滅する。

コードの入力手順

1. App StoreまたはGoogle PlayでmenuアプリをDL

2. メールアドレスで新規会員登録

3. トップ画面右下の「マイページ」をタップ

4. 「クーポン・ポイント」→「プロモーションコード」を選択

5. コードを入力して「確認する」をタップ

6. 注文時に割引が自動適用される

注文確定後にコードを入力しても適用されない。必ず注文前に入力しておくこと。

適用条件と注意点

  • 新規ユーザーのみ対象（既存ユーザーは使用不可）
  • 1回あたり1,500円以上の注文が必要
  • コード登録後31日以内に1回目の注文を完了すること
  • 現金払い不可（クレジットカード・PayPay等の電子決済のみ）
  • 一部の店舗・商品は対象外になる場合あり
  • 1アカウントにつき1コードのみ適用可能

併用できるキャンペーン

menu passというサブスク会員（月額480円）を無料トライアル期間中に利用すると、配送料が無料になる。プロモーションコードの割引と配送料無料が重なる期間が最もお得だ。

また、menu独自のポイントプログラムも存在し、注文額に応じてポイントが貯まる。ポイントは次回の注文に使えるため、実質的な割引率がさらに上がる。

PayPayやd払いで支払うと、各決済サービス側のキャンペーンも重なる場合がある。支払い方法を選ぶ前にキャンペーン情報を確認しておくと損がない。

お得に使うコツ

1,500円ちょうどで注文するより、1,700〜2,000円分まとめて頼んだほうが1回あたりの割引率が高くなる。送料分を考えると、少し多めに頼んでまとめ注文にするのが基本戦略だ。

8回の割引を使い切るまでの31日間は、外食やテイクアウトをできるだけmenuに集約すると効率がいい。週2回ペースで使えば1ヶ月以内に消化できる計算になる。

初回は1,200円OFFと最大の割引なので、やや高めのレストランを試してみるのも手だ。普段なら頼まないような少し良い店を、実質無料に近い形で体験できる。

プロモーションコード

RNPA_NYVD96

上記のコードをアプリの「プロモーションコード」欄に入力して使用する。新規登録後すぐに入力しておくと、使い忘れを防げる。
