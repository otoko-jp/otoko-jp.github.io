Uber Eatsとは

フードデリバリーサービス。対応エリア内のレストランから注文すると最短30分程度で届く。アカウント登録はメールアドレスのみで完結し、本人確認は不要だ。

国内の対応エリアは主要都市を中心に拡大しており、深夜でも注文できる店舗が多い。ランチからディナー、コンビニ商品まで幅広く対応している。

特典の詳細

プロモーションコードを使うと、初めてUber Eatsを利用する際に初回から4回の注文がそれぞれ最大1,500円OFFになる。合計で最大6,000円分の割引だ。

特典金額条件
新規ユーザーの割引初回から4回 ¥1,500 OFF¥2,000以上の注文・28日間有効
紹介した側の報酬次回注文 ¥1,300 OFF¥1,800以上の注文・14日間有効

コードの入力手順

1. Uber Eatsアプリをインストール（App Store / Google Play）

2. メールアドレスでアカウント登録

3. 画面下の「アカウント」→「プロモーション」をタップ

4. 「コードを追加」にプロモーションコードを入力

5. 「追加」をタップして完了

注文前に入力すること。注文後は適用不可。

適用条件と注意点

  • 初めてUber Eatsを利用する方のみ対象
  • ¥2,000以上の配達注文に適用（現金払い・一部店舗は除外）
  • コード入力後28日以内に4回使い切ること
  • 同一アカウントへの複数コード適用は不可

お得に使うコツ

まとめ注文で1回あたりの割引を最大化する。¥2,000ちょうどで注文するより、¥2,500〜¥3,000分まとめて頼んだ方が送料との兼ね合いでお得になりやすい。

Uber One（月額会員）の無料トライアル期間中と併用するとさらにお得。配達手数料が無料になるため、プロモーションコードとの組み合わせが最も恩恵が大きい。

混雑時間帯はサービス料が上乗せされるため、昼・夕方の注文の方が¥2,000の最低ラインを超えやすい。
