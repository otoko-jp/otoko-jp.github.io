## プロモーションコード

**eats-k3ysylxkm9**

初めてUber Eatsを使う方限定。このコードを入力すると**初回から4回の注文が¥1,500 OFF**になる。最大で6,000円分お得。

| 特典 | 金額 | 条件 |

| 友達（あなた）の割引 | 初回から4回 ¥1,500 OFF | ¥2,000以上の注文・28日間有効 |
| 紹介した側の報酬 | 次回注文 ¥1,300 OFF | ¥1,800以上の注文・14日間有効 |

## コードの入力手順

1. Uber Eatsアプリをインストール（App Store / Google Play）
2. アカウント登録（メールアドレスのみでOK）
3. 画面下の「アカウント」→「プロモーション」をタップ
4. 「コードを追加」に `eats-k3ysylxkm9` を入力
5. 「追加」をタップして完了

**注文前に入力すること。注文後は適用不可。**

## 適用条件と注意点

- 初めてUber Eatsを利用する方のみ対象
- ¥2,000以上の配達注文に適用（現金払い・一部店舗は除外）
- コード入力後28日以内に4回使い切ること
- 同一アカウントへの複数コード適用は不可
